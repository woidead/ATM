# ATM

## INSTALLATION
`git clone https://github.com/woidead/ATM`
`cd ATM`

## CREATE AND ACTIVATE VENV
`python3 -m venv env`
`source env/bin/activate`

## REQUIREMENTS INSTALLATION
`pip install -r requirements.txt`

## SET CONNECTION
`touch .env`
PUT THE CONNECTION PARAMETS IN `.env`


## START
`python3 main.py`
